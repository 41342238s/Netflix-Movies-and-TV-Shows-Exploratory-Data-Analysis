Netflix Movies and TV Shows — Exploratory Data Analysis¶

Data source: Netflix Movies and TV Shows (Kaggle)

Objectives¶

  1. How have the numbers of Movies and TV Shows added to Netflix changed by release year, and has the balance between them shifted over time?
  2. What are the top 10 most common genres on Netflix?

Analysis Plan¶

1. Load data:
    Read netflix_titles.csv

2. Inspect and clean:
    Check the number of missing values in each column
    Check for duplicate rows

3. Group and analyze:
    Count Movies and TV Shows for each release year
    Split the listed_in column (a title can have multiple genres) and count how many times each genre appears
    Find the top 10 most common genres

4. Visualize:
    Line chart: number of Movies vs TV Shows by release year
    Bar chart: top 10 genres

5. Interactive feature
    
6. Summarize:
    Describe the trends observed, answering the objectives above

# Objective 1: count Movies and TV Shows for each release year
movie_counts = {}
tv_counts = {}

for i in range(len(df)):
    year = df["release_year"][i]
    show_type = df["type"][i]

    if show_type == "Movie":
        if year in movie_counts:
            movie_counts[year] = movie_counts[year] + 1
        else:
            movie_counts[year] = 1
    else:
        if year in tv_counts:
            tv_counts[year] = tv_counts[year] + 1
        else:
            tv_counts[year] = 1

# Sort by year so the line chart draws in order
movie_years = sorted(movie_counts.keys())
movie_values = []
for year in movie_years:
    movie_values.append(movie_counts[year])

tv_years = sorted(tv_counts.keys())
tv_values = []
for year in tv_years:
    tv_values.append(tv_counts[year])

# Objective 2: split the listed_in column and count how many times each genre appears
genre_counts = {}

for i in range(len(df)):
    genres = df["listed_in"][i]
    if pd.isna(genres):
        continue
    genre_list = genres.split(", ")

    for genre in genre_list:
        if genre in genre_counts:
            genre_counts[genre] = genre_counts[genre] + 1
        else:
            genre_counts[genre] = 1

# Turn the dictionary into a list of (count, genre) pairs, then sort it
genre_count_list = []
for genre in genre_counts:
    genre_count_list.append((genre_counts[genre], genre))

genre_count_list.sort(reverse=True)
top_10_genres = genre_count_list[:10]

## Step 5: Interaction

# Allow users to enter the year they want to search for
user_input = input("Please enter the year (before 2020) of release you wish to search for (e.g.: 2020): ")

try:
    # Convert the entered text to an integer
    search_year = int(user_input)
    
    # Filter the data for that year
    result = df[df['release_year'] == search_year]
    
    # Check for data
    if len(result) > 0:
        print(f"\n--- Search Results: {search_year} There were a total of  {len(result)} works in this year ---")
        
        # Count the number of movies and TV shows separately
        type_counts = result['type'].value_counts()
        print(f"incluging: {type_counts.get('Movie', 0)} movies, {type_counts.get('TV Show', 0)} TV Show\n")
        
        # Show the titles and descriptions of the first 10 works
        print("【10 Randomly Recommended Films from That Year】:")
        display(result[['title', 'type', 'listed_in']].head(10))
    else:
        print(f"Sorry, nothing was found in the database from {search_year} in our database.")
        
except ValueError:
    print("Invalid input format. Please make sure you entered a numeric year.（e.g.：2019）。")

Step 6: Conclusion¶

  1. How has the number of new movies and TV shows added to Netflix changed over the years? Has the ratio between the two shifted over time? Trends in Movies and TV Shows: Based on historical data, the number of movies released each year has been significantly higher than that of TV shows.
    Recent Changes: While the number of movies released has seen a sharp spike in recent years, followed by a downward trend, the number of TV shows released has continued to grow steadily. The gap between the two has narrowed significantly, indicating that Netflix’s catalog is gradually shifting toward more episodic content.

  2. What are the top 10 most common genres on Netflix? Popular Genres: International films are the most prevalent genre on the platform, with a total of 2,752 titles, followed closely by dramas and comedies.
    Genre Breakdown: The top ten list covers a diverse range of content types, including both film-centric categories (e.g., independent films, action and adventure, romance) and television categories (e.g., international TV series, TV series). Documentaries and children’s and family films also account for a significant portion of the platform’s content.
